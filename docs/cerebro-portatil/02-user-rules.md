# Camada 2 — User Rules da conta Cursor

## O que é

Regras pessoais do Gabriel que ficam atreladas à **conta Cursor**, não ao projeto. Sincronizam automaticamente entre os 3 PCs (branco, preto, notebook) quando o Gabriel logar com a mesma conta.

## Onde configurar

1. Abre Cursor
2. `Ctrl+Shift+P` → `Cursor Settings`
3. Aba **Rules** → seção **User Rules**
4. Cola o conteúdo abaixo

## Por que é portátil

Cursor sincroniza User Rules via conta. Só precisa estar logado com mesma conta nos 3 PCs.

## Verificar sincronização

Em cada PC, abre `Cursor Settings → Rules → User Rules` e confirma que o conteúdo é o mesmo.

## Exemplo do dia-a-dia

É igual o cartão do banco. Você não precisa carregar o dinheiro entre as agências — chega em qualquer agência, passa o cartão e o sistema lembra de você. Tanto faz se a agência tá em Goiânia ou em Patrocínio: o banco é o mesmo, a conta é a mesma.
